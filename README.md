# PEE15 Module 2 Reviewer Website

Static reviewer and practice website for **PEE15-M: Load Characteristics**.

## Pages

- `index.html` - full lesson reviewer (content, formulas, examples)
- `practice-website.html` - MCQ, identification, and problem-solving drills

## GitHub Pages Deployment

1. Create a GitHub repository and upload these files to the repository root.
2. In GitHub, open:
   - `Settings` -> `Pages`
3. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or `master`)
   - **Folder**: `/ (root)`
4. Save and wait for deployment.
5. Open the published URL shown in Pages settings.

## Notes

- This project is plain HTML/CSS/JS; no build step is required.
- `index.html` is the homepage used by GitHub Pages.
- `practice-website.html` is linked directly from `index.html`.
- `.nojekyll` is included to prevent Jekyll processing issues.

## Local Preview

You can open `index.html` directly in your browser.
